1. **Clone the Repository**
    ```sh

    cd your-blog-repo
    ```

2. **Backend Setup**
    ```sh
    cd backend
    pip install -r requirements.txt
    python manage.py migrate
    python manage.py createsuperuser
    python manage.py runserver
    ```

3. **Frontend Setup**
    ```sh
    cd frontend
    npm install
    npm start
    ```
